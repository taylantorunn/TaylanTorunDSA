# Flight Delay Prediction Using Machine Learning

**Project Description:**

In this project, I aim to analyze airline flight data to understand the factors that contribute to flight delays and to build a machine learning model that can predict whether a flight will be delayed. Flight delays are a common issue in the aviation industry, affecting both operational efficiency and customer satisfaction. By identifying key factors behind delays, this project seeks to provide insights that could help airlines improve scheduling and decision-making processes.  

The dataset used in this project is the “PIA 2026 Smart Flight Dataset,” which includes information such as flight schedules, route details, weather conditions, passenger load, and delay status. Since the dataset contains multiple relevant variables, it is well-suited for both exploratory data analysis (EDA) and predictive modeling. Additionally, I plan to enrich the dataset by creating new features such as time-of-day categories (morning, afternoon, evening) and distance-based route classifications (short, medium, long haul), which may improve the performance of the model.  

The project will follow the standard data science pipeline. First, the data will be cleaned and prepared by handling missing values and formatting variables appropriately. Then, exploratory data analysis will be conducted to identify patterns and relationships between variables such as weather conditions, departure times, and delays. Based on these insights, relevant features will be selected for modeling.  

For the predictive task, a classification approach will be used, where the target variable will be whether a flight is delayed or not. I plan to implement Logistic Regression as a baseline model and potentially compare it with another method such as Random Forest. The performance of the model will be evaluated using metrics such as accuracy and F1-score.  

The expected outcome of this project is a better understanding of the key drivers of flight delays and a predictive model that can estimate delay likelihood. These findings could be useful for airlines to optimize operations and enhance customer experience.
